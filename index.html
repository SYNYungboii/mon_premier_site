HTTPS Website

Le HTTPS signifie « HyperText Transfer Protocol Secure »

C'est un protocole de communication qui permet de sécuriser les échangesde données entre un serveur et un client.

Il est notamment utilisé pour le commerce en ligne afin de protéger les informationssensibles des utilisateurs, comme le numéro de leur carte de crédit et permet de verifier l'identité du site Web auquel il accède.

HTTPS est aussi utilisé pour la consultation de données privées, comme les mails

Le HTTPS a été créé en 1994 par Netscape.

La securité du HTTP est basée sur un algorithme et sur la reconnaissance de validité du certificat d'authentification du site visité.
